AI Helper - React & Google Gemini Integration
A modern, responsive web application built with React that leverages the power of Google's Gemini 3.1 Flash/Pro AI to provide intelligent text analysis and assistance.


🛠️ Tech Stack
Frontend: React.js

Styling: CSS3 (with Custom Properties & Flexbox)

API: Google Generative AI (Gemini API)

Icons/Fonts: Google Fonts (Inter)
